# Кубики и числа 6×6
Что нужно
3 игральных кубика, таблица 6×6 с числами от 1 до 36, фишки, 2–4 игрока.
Ход игрока
Игрок бросает все 3 кубика. Затем берёт любые из выпавших чисел — одно, два или все три — и составляет из них выражение со знаками + и ×. 
Например, выпали числа 1, 2, 5. 
Можно закрыть 1 или 2, или 5. Можно закрыть 1+2=3 или 2×5=10. Можно 1+2×5=11, а можно (1+2) ×5=15
Результат называет вслух вместе с объяснением, например: «Получаю 15, потому что (1+2)×5». Остальные проверяют. Если верно, число закрывается фишкой (монетой) в таблице. Если игрок не может составить ни одного свободного числа, он пропускает ход.
Победа
Побеждает тот, кто первым объявляет полосу из 4 зачёркнутых чисел подряд — по горизонтали, вертикали или диагонали (в любом направлении). Полоса может состоять из чисел, зачёркнутых любыми игроками. При объявлении нужно назвать все 4 числа вслух.
